# ClipForge AI

Open Source AI Video Clipping Platform.

## Features

- Video Upload
- AI Transcription
- AI Clip Detection
- Subtitle Generator
- FFmpeg Rendering
- Railway Deployment

## Tech Stack

- FastAPI
- Next.js
- PostgreSQL
- Redis
- Celery
- Faster-Whisper
- FFmpeg

## Run

```bash
docker compose up --build
```
